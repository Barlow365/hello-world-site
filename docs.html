Quick start

Install the SDK and follow these quick steps to send your first transaction and register a membership.

npm install @the-grid/sdk
// then import and initialize the client

SDK & API reference

Our SDK is small and composable. The reference below shows key primitives used by production apps.

  • Authentication — wallet-based auth + session management
  • Memberships — onchain membership objects and access control
  • Payments — low-fee settlement flows and revenue splits
  • Content — onchain content pointers and licensing primitives
